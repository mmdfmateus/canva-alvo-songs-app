## ğŸ“ 1. App Icon (512x512 pixels, PNG)

### Technical Requirements
- **Size**: Exactly 512x512 pixels
- **Format**: PNG (with transparency preferred)
- **File Size**: Under 1MB (ideally under 500KB)
- **Color Mode**: RGB

### Design Specifications

**Design Concept:**
Create an icon that represents music/lyrics and slides/presentations. The icon should be:
- **Recognizable** at small sizes (appears as 64x64 in some contexts)
- **Simple** and uncluttered
- **Professional** and modern
- **On-brand** with your app's theme

**Recommended Design Elements:**

**Option 1: Music Note + Slides**
```
[Musical Note Icon] + [Document/Slide Icon]
- Combine a musical note with a document or slide icon
- Use your brand colors (orange/white or complementary colors)
- Clean, flat design
```

**Option 2: Stylized "A" for Alvo**
```
[Letter "A" with music elements]
- Modern, geometric "A"
- Subtle music note or wave pattern
- Professional typography
```

**Option 3: Microphone + Text Lines**
```
[Microphone Icon] + [Text Lines/Slides]
- Microphone representing songs/music
- Text lines representing lyrics/slides
- Clean, minimalist design
```

**Option 4: Simple Music Note**
```
[Large, Stylized Music Note]
- Single, prominent music note
- Modern, geometric style
- Vibrant background color
```

### Color Palette Suggestions

**Primary Option (Orange/White Theme):**
- Background: `#FF6B35` (Orange) or `#FFFFFF` (White)
- Accent: `#FFFFFF` (White) or `#FF6B35` (Orange)
- Text/Icon: High contrast color

**Alternative Options:**
- Blue/Purple gradient
- Teal/White
- Dark background with light icon

### Tools for Creation

**Recommended Tools:**
1. **Canva** (easiest)
   - Go to Canva.com
   - Create custom size: 512x512
   - Use icon templates
   - Export as PNG

2. **Figma** (professional)
   - Create 512x512 frame
   - Design icon
   - Export as PNG @2x

3. **Adobe Illustrator**
   - Create 512x512 artboard
   - Design vector icon
   - Export as PNG

4. **Online Icon Generators**
   - IconKitchen
   - AppIcon.co
   - MakeAppIcon

### Step-by-Step: Creating Icon in Canva

1. **Create New Design**
   - Go to canva.com
   - Click "Create a design" â†’ "Custom size"
   - Set: 512px Ã— 512px

2. **Choose Background**
   - Solid color or subtle gradient
   - Consider your brand colors

3. **Add Icon Elements**
   - Search for "music note" icons
   - Search for "document" or "slide" icons
   - Combine elements creatively

4. **Add Text (Optional)**
   - If including text, use large, bold font
   - Ensure readability at small sizes
   - Consider just "A" for Alvo

5. **Finalize Design**
   - Ensure all elements are centered
   - Check contrast and readability
   - Remove unnecessary elements

6. **Export**
   - Click "Download"
   - Choose "PNG"
   - Select "Transparent background" if applicable

### Icon Design Checklist

- [ ] Exactly 512x512 pixels
- [ ] PNG format
- [ ] Recognizable at 64x64 size
- [ ] High contrast (readable on light and dark backgrounds)
- [ ] No text or very large, simple text
- [ ] Professional appearance
- [ ] Matches app theme
- [ ] File size under 1MB

## ğŸ“¸ 3. Screenshots (3-5 images)

### Technical Requirements
- **Sizes**: Various (Canva accepts different sizes)
- **Format**: PNG or JPG
- **Recommended**: 1280x720, 1920x1080, or match your app's aspect ratio
- **File Size**: Under 2MB each

### Screenshot 1: Main Interface

**What to Show:**
- Song browser with list of songs
- Search bar visible
- Clean, professional interface
- A few songs visible in the list

**How to Capture:**
1. Open your app in Canva
2. Make sure songs are loaded
3. Take screenshot of the main interface
4. Crop to focus on the app UI
5. Add subtle annotation if helpful (e.g., "Search songs")

**Tips:**
- Use actual app screenshot (not mockup)
- Ensure good lighting/contrast
- Remove any personal information
- Show the app in a clean state

### Screenshot 2: Settings Panel

**What to Show:**
- Settings panel expanded
- Color selectors visible
- Lines per slide configuration
- All customization options

**How to Capture:**
1. Open app
2. Click "Settings" to expand
3. Show color selectors and number inputs
4. Take screenshot
5. Add annotation: "Customize colors and layout"

**Tips:**
- Show settings in an organized way
- Highlight key customization options
- Ensure text is readable

### Screenshot 3: Slide Generation Progress

**What to Show:**
- Progress bar visible
- "Adding pages: X of Y" text
- Loading state
- Real-time feedback

**How to Capture:**
1. Select a song
2. Wait for progress indicator to appear
3. Capture during slide generation
4. Show the progress bar clearly

**Tips:**
- Show actual progress (not 0%)
- Demonstrate real-time feedback
- Highlight the progress indicator

### Screenshot 4: Final Result - Completed Slides

**What to Show:**
- Multiple completed slides in Canva
- Title slide visible
- Lyrics slides visible
- Professional formatting

**How to Capture:**
1. Generate slides for a song
2. Show the Canva presentation with slides
3. Include 3-4 slides in view
4. Show title slide and lyrics slides

**Tips:**
- Use a song with multiple slides
- Show variety (title + lyrics slides)
- Ensure slides look professional
- Highlight the final result

### Screenshot 5: Search Functionality

**What to Show:**
- Search bar with query
- Filtered results
- Search in action

**How to Capture:**
1. Type a search query
2. Show filtered results
3. Highlight the search feature

**Tips:**
- Show actual search results
- Demonstrate filtering works
- Keep search query simple

### Screenshot Guidelines

**General Tips:**
- Use actual app screenshots (not mockups)
- Ensure consistent styling across all screenshots
- Add subtle annotations if helpful (arrows, labels)
- Remove any personal/sensitive information
- Ensure text is readable
- Use consistent background/theme

**Annotation Tools:**
- Canva (add arrows, text, highlights)
- Figma (add annotations)
- Snagit or similar tools

**Screenshot Checklist:**
- [ ] 3-5 screenshots prepared
- [ ] All show actual app functionality
- [ ] Text is readable
- [ ] Consistent styling
- [ ] No personal information
- [ ] Professional appearance
- [ ] Each under 2MB
